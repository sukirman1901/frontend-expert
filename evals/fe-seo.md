# E17 — FE SEO minimum

## Prompt

```text
Landing page marketing “Acme — ship faster”. Public, shareable. Langsung saja.
```

## Expected

1. Loads `fe-seo`
2. Unique `<title>` + meta description
3. One H1 aligned with title
4. OG title/description (image optional if noted as TODO with real placeholder path)
5. Not a blank client-only shell without content for crawlers when claiming SEO

## Pass criteria

- [ ] Title + description present
- [ ] One h1
- [ ] OG basics or explicit next step for image
- [ ] Mentions `fe-seo` or SEO checklist

## Fail if

Same generic title only, multi-h1 hero spam, or no meta on a public landing ask.
